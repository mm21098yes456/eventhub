  EventHub

Sistema web de gestión de eventos desarrollado con HTML, CSS y JavaScript.

//Características//

* CRUD de eventos
* Dashboard dinámico
* Buscador de eventos
* Filtro por categorías
* Modo oscuro
* API REST del clima
* Geolocalización
* Web Workers
* Exportar PDF
* Login básico
* Responsive Design
* LocalStorage y SessionStorage
* Gráficos con Chart.js
* SweetAlert2
* Animaciones CSS

Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Chart.js
* SweetAlert2
* jsPDF
* Open Meteo API

# Funcionalidades

## CRUD

Permite crear, editar y eliminar eventos.

## Dashboard

Muestra:

* Total de eventos
* Eventos próximos
* Eventos finalizados

## API del clima

Obtiene el clima en tiempo real utilizando geolocalización y fetch API.

## Modo oscuro

Permite cambiar entre tema claro y oscuro.

## Exportar PDF

Genera un PDF con la lista de eventos registrados.

## Login básico

Sistema simple de autenticación utilizando SessionStorage.
